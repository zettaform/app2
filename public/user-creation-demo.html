‚ÑπÔ∏è API Information

Endpoint:

GET /api/external/users

Required Parameters:

  • first_name - User's first name
  • last_name - User's last name
  • email - User's email address
  • password - User's password (min 6 chars)
  • x-admin-key - Admin global key for authentication

Optional Parameters:

  • role - User role (user, manager, admin) - Default: user
  • status - Account status (active, inactive, suspended, pending, blocked) - Default: active
  • avatar - Avatar name (goku, vegeta, etc.) - Default: goku
